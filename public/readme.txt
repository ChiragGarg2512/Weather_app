This folder contains all the css, img and js files for background work.
1) Css files have stylescript
2) Image has the image to be rendered
3) js file is shown in the script when the page is inspected


In js\app.js file,
    This file works after being called in templates\views\index.js as <script src ="/js/app.js"></script>.
    The functions running in app.js will show their output in the console tab of inspect of the web page of the browser.
